### Counter() [猬锔](#main)
```python

d = Counter(a=1, b=2)
c + d                       # dodawanie 
#Output: Counter({'a': 4, 'b': 3})
c - d                       # odejmowanie, zachowanie tylko dodatnich liczb
#Output:Counter({'a': 2})
c & d                       # min ka偶dego klucza 
#Output:Counter({'a': 1, 'b': 1})
c | d                       # max ka偶dego klucza
#Output:Counter({'a': 3, 'b': 2})
```
<h5> Metody </h5>

   * __.elements()__ -> Zwracan iteracje kluczy: 'klucz'*vaule
   * __.most_common()__ -> zwraca liste krotek malejca wedug wartoci kluczy
   * __.subtract()__ -> zwraca counter jako wynik odejmowania z zachowaniem ujemnych wartoci
